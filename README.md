## 🎯 About

Py.repo is a collection of Python projects that demonstrate various programming concepts, from basic games to data analysis scripts. Each project is self-contained with clear documentation and easy-to-understand code.

---

## 📦 Projects

### 🎮 Games
| Project | Description | Concepts |
|---------|-------------|----------|
| Number Guessing Game | Guess the number with hints | Random, Loops, Conditionals |
| Rock Paper Scissors | Classic game vs computer | Random, Dictionaries |
| Tic Tac Toe | Two-player game | Lists, Functions |

### 🛠️ Utilities
| Project | Description | Concepts |
|---------|-------------|----------|
| Password Generator | Secure password creation | Random, String |
| Calculator | Basic math operations | Functions, OOP |
| Todo App | Task management | File I/O, Lists |

### 📊 Data Analysis
| Project | Description | Concepts |
|---------|-------------|----------|
| CSV Analyzer | Parse and analyze CSV files | Pandas, Statistics |
| Web Scraper | Extract data from websites | BeautifulSoup, Requests |
| API Client | Fetch and process API data | Requests, JSON |

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/Ankitavasudev/Py.repo.git

# Navigate to directory
cd Py.repo

# Run any project
python number_guess.py
```
